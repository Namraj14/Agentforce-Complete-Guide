# Agentforce Interview Questions

## Q1 What is Agentforce?

Agentforce is Salesforce's AI Agent platform that uses LLMs, grounding, trust layer, and actions to automate customer and employee interactions.

---

## Q2 What are the core components of Agentforce?

- Agent
- Topics
- Instructions
- Actions
- Grounding
- Trust Layer
- LLM

---

## Q3 What is Grounding?

Grounding provides trusted business data to the LLM to reduce hallucinations.

---

## Q4 Difference between Agentforce and Einstein Bots?

Einstein Bots are rule-based.

Agentforce uses AI reasoning and dynamic actions.

---

## Q5 What is an Agent Action?

An executable capability that allows an agent to perform tasks.

Examples:

- Flow
- Apex
- Prompt
- API

---

## Q6 Explain Agentforce architecture.

User → Agent → Topic → Instructions → Grounding → Actions → LLM → Response
