# What is an LLM?

LLM stands for Large Language Model.

A Large Language Model is trained on massive amounts of text data to understand and generate human language.

## Examples

- OpenAI GPT
- Claude
- Gemini

## Capabilities

- Question Answering
- Summarization
- Translation
- Content Generation
- Reasoning

## How LLM Works

Input
→ Tokenization
→ Model Processing
→ Prediction
→ Output

## Agentforce

Agentforce uses LLMs to understand user requests and generate responses.
